# Segmento SDK

This project is a TypeScript SDK that will be uploaded as an NPM package. It consists of 2 packages:

- referral - this package has functionality to sign a message using Solana wallet and it can call backend function with telegram, email, wallet_address and signature to create a referral code that is then returned by the backend
- ui - this is a pure html, css, javascript implementation of a modal where user can sign message, enter telegram and email. It should use shadow root.
